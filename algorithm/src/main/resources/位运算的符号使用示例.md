## 判断奇偶性：
x & 1 == 1 表示 x 是奇数 (最低位为 1)。
x & 1 == 0 表示 x 是偶数 (最低位为 0)。
## 判断第 i 位是否为 1：
x & (1 << i) 如果结果不为 0，则表示 x 的第 i 位（从右往左，从 0 开始数）是 1。否则是 0。
## 将第 i 位设置为 1：
x \| (1 << i) 可以将 x 的第 i 位变为 1，其他位不变。
## 将第 i 位设置为 0：
x & ~(1 << i) 可以将 x 的第 i 位变为 0，其他位不变。（先构造一个第 i 位为 0，其余位为 1 的掩码 ~(1 << i)，然后与 x 进行 & 运算）。
## 将第 i 位取反：
x ^ (1 << i) 可以将 x 的第 i 位取反，其他位不变。
## 清零最低位的 1：
x & (x - 1) 可以将 x 的二进制表示中最右边的那个 1 变成 0。常用于计算一个数的二进制中有多少个 1。
## 获取最低位的 1：
x & (-x) 可以得到 x 的二进制表示中最低位那个 1 所代表的数值。例如，x = 12 (1100)，-x = -12 (...11110100)，x & (-x) 结果是 4 (0100)。
## 交换两个数（不使用临时变量）：
a ^= b; b ^= a; a ^= b; 利用 x ^ x = 0 和 x ^ 0 = x 的性质。 (注意：如果 a 和 b 是同一个变量，这种方法会出错)
